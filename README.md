## 📌 About Vingo

Vingo is a **full-stack food delivery web application** built using the MERN stack.
It allows users to order food online, shop owners to manage orders, and delivery boys to handle deliveries in real-time.

---

## 👨‍💻 Key Features

### 🧑‍🍳 Customer (User)

* Browse food items 🍕
* Add to cart 🛒
* Place orders ✅
* Real-time order tracking 📍

### 🏪 Shop Owner

* Manage food items 🍔
* Accept / reject orders
* Update order status

### 🚴 Delivery Boy

* Receive delivery tasks
* Update delivery status
* Real-time updates using Socket.IO

---

## ⚙️ Tech Stack

* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Realtime:** Socket.IO
* **Authentication:** JWT / Passport.js

## 🖥️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/princekumar-995/8.Vingo.git
cd 8.Vingo
```

### 2️⃣ Install Dependencies

```bash
cd backend
npm install

cd ../frontend
npm install
```

### 3️⃣ Run Project

```bash
# Backend
cd backend
npm start

# Frontend
cd frontend
npm run dev
```
